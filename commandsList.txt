anagramma - анаграмма (анаграмм?)
association - ассоциации (дом любовь)
phrase - фразеологизмы (камен<ный цветок>)
mask - маска (м?ск*)
rebus - ребус (шир а -> широта)
plus - плюсограмма (мода => дом камод)
meta - метаграммы (тесто => место тесть)
brukva - брюквы (ветка => вето клетка)
logo - логогрифы (пест=перст, вера=венера)
any - все форматы (ком => лом слом ом)
dick - расчлененки (домашка3 колено2)
cross - общая часть (стер кроб = мистер микроб)
maskword - маска + слово (за?ка = заКЛАДка)
book - поиск по книгам (мызагран - ведьмы за границей)
film - поиск по фильмам (кийоре - крепкий орешек)
painting - поиск по картинам (похищение - Похищение Европы)
roman - римские числа (DC - 600, 600 - DC)
adj - прилагательные к существительному (дом - доходный издательский)
noun - существительные к прилагательному (доходный - дом бизнес)
gapoifika - ГаПоИФиКа
ss - системы счисления
notes - перевод нот в цифры/буквы (до ля ми - ...)
associationlogic - ассоциации + логика (монарх + плавание = король кроль)
subword - подслово (колодец => кол)
longword - надслово (колода => киноплощадка)
azimut - расчет координат (lat lon ang dist)
noize - песни Noize MC (название / подстрока)
sborka - сборка (дом из сои кокоса 1 1 1 1 - диск)
sborkaline - сборка строки (дом из сои кокоса 1 4 6 9 - диск)
auto - переключение автоматических команд
